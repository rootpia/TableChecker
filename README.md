# TableChecker
TABLEをチェックするためのChrome拡張機能

## install
1. リポジトリをクローン
```
$ git clone https://github.com/rootpia/TableChecker.git
```

1. Chromeを起動し「右上の\[：] > 拡張機能 > 拡張機能を管理」を選択
　→拡張機能ページが表示される

1. 拡張機能ページの「デベロッパーモード」のトグルをONにする

1. 拡張機能ページの「パッケージ化されていない拡張機能を読み込む」を押下
　→フォルダ選択ダイアログが表示される

1. クローンしたフォルダを選択し、\[フォルダを選択]ボタンでダイアログを閉じる
　→拡張機能が追加される

1. ブラウザのURL表記右にある「拡張機能」のアイコンを押下
　→有効な拡張機能の一覧がポップアップで表示される

1. \[TIME Checker]をピン止めする
　→拡張機能アイコンの横にTime Checkerアイコンが表示される


## Usage
1. ブラウザでチェック対象ページを表示した状態で、Time Checkerアイコンを押下
　→実行条件のポップアップ表示

1. 実行条件を設定し、\[チェック実行]ボタンを押下
　→チェック結果がポップアップに表示され、表示中のテーブルの整合性エラー行、該当セルが赤色になる
